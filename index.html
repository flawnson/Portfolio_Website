Molecule Melting point regression

with a basic ANN

Overview

  • Date : October, 2018

I built a melting point temperature predictor in Python using the Pytorch library and NumPy, as well as Pandas Python packages. All data preprocessing, model architecture, checkpoints, as well as predictions are within the melting.py file. The model receives padded, normalized, and integerized SMILES strings as input to output a predicted melting point. The dataset consists of a couple thousand labeled datapoints in a csv file format. Molecules included those that are not metallic.

Carbon Molecule Autoencoder

with Keras

Overview

  • Date : November, 2018

My first autoencoder built with Keras and trained on an open source collection of 12K datapoints filtered to contain only molecules with at least one carbon atom in them. The idea was to learn a latent space that would hold information that made carbon based molecules unique in properties.
